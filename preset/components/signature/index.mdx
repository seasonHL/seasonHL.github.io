要自定义这个组件，你需要先用 Figma 或其他矢量图编辑器创建一个 SVG。该 SVG 需要包含一个或多个带 `stroke` 属性的 `<path>` 元素，因为动画效果仅作用于这些描边路径。你可以参考[这个 SVG](https://github.com/zhuozhiyongde/Arthals-Ink/blob/main/src/assets/signature.svg)。

此外，你还可以使用一些附加属性来自定义组件：

1. `data-duration`（可选）：动画时长（毫秒）。未指定时会使用 `defaultDuration` 的值。例如：`<path data-duration="800" ... />`
2. `seq`（可选）：路径播放顺序。未指定时按 SVG 文件中的出现顺序播放。例如：`<path seq="1" ... />`、`<path seq="2" ... />`。

默认情况下，组件会从 `src/assets/signature.svg` 加载 SVG 文件。请将准备好的文件放到该路径下，并命名为 `signature.svg`。

## 组件 Props

使用组件时，你可以传入下列属性来调整动画行为：

| Prop              | Type                         | 说明                                                                         | 默认值      |
| ----------------- | ---------------------------- | ---------------------------------------------------------------------------- | ----------- |
| `mode`            | `'loop'` \| `'once-on-view'` | 动画模式。`loop`：持续绘制并回退；`once-on-view`：进入视口后播放一次。        | `'loop'`    |
| `drawDelay`       | `number`                     | （循环模式）绘制下一条路径前的延迟毫秒数。                                   | `200`       |
| `rollbackDelay`   | `number`                     | （循环模式）回退上一条路径前的延迟毫秒数。                                   | `80`        |
| `loopPause`       | `number`                     | （循环模式）每次完整循环结束后的暂停毫秒数。                                 | `2000`      |
| `threshold`       | `number`                     | （视口触发模式）触发动画的可见度阈值（0.0 到 1.0）。                          | `1.0`       |
| `defaultDuration` | `number`                     | 每条笔画默认动画时长（毫秒）。                                                | `600`       |
| `class`           | `string`                     | 附加到组件根元素的 CSS class。                                                | `undefined` |
